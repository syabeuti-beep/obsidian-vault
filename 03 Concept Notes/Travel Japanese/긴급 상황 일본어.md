# 긴급 상황 일본어

상위 노트: [[여행 일본어 회화]]
관련 노트: [[길 묻기와 교통 일본어]], [[숙소와 체크인 일본어]]

## 바로 써야 하는 말

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 도와주세요 | 助けてください。<br>(타스케테 쿠다사이) | 타스케테 쿠다사이 |
| 경찰을 불러주세요 | 警察を呼んでください。<br>(케이사츠오 욘데 쿠다사이) | 케이사츠오 욘데 쿠다사이 |
| 구급차를 불러주세요 | 救急車を呼んでください。<br>(큐-큐-샤오 욘데 쿠다사이) | 큐-큐-샤오 욘데 쿠다사이 |
| 병원에 가고 싶습니다 | 病院に行きたいです。<br>(뵤-인니 이키타이 데스) | 뵤-인니 이키타이 데스 |
| 길을 잃었습니다 | 道に迷いました。<br>(미치니 마요이마시타) | 미치니 마요이마시타 |
| 휴대폰을 잃어버렸습니다 | 携帯をなくしました。<br>(케이타이오 나쿠시마시타) | 케이타이오 나쿠시마시타 |
| 지갑을 잃어버렸습니다 | 財布をなくしました。<br>(사이후오 나쿠시마시타) | 사이후오 나쿠시마시타 |

## 몸 상태 말하기

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 아픕니다 | 痛いです。<br>(이타이 데스) | 이타이 데스 |
| 머리가 아픕니다 | 頭が痛いです。<br>(아타마가 이타이 데스) | 아타마가 이타이 데스 |
| 배가 아픕니다 | お腹が痛いです。<br>(오나카가 이타이 데스) | 오나카가 이타이 데스 |
| 열이 있습니다 | 熱があります。<br>(네츠가 아리마스) | 네츠가 아리마스 |
| 약이 필요합니다 | 薬が必要です。<br>(쿠스리가 히츠요- 데스) | 쿠스리가 히츠요- 데스 |
| 알레르기가 있습니다 | アレルギーがあります。<br>(아레루기-가 아리마스) | 아레루기-가 아리마스 |

## 말이 안 통할 때

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 일본어를 잘 못합니다 | 日本語があまり話せません。<br>(니혼고가 아마리 하나세마셍) | 니혼고가 아마리 하나세마셍 |
| 한국어 가능한 분 있나요? | 韓国語を話せる人はいますか。<br>(칸코쿠고오 하나세루 히토와 이마스카) | 칸코쿠고오 하나세루 히토와 이마스카 |
| 영어 가능한 분 있나요? | 英語を話せる人はいますか。<br>(에-고오 하나세루 히토와 이마스카) | 에-고오 하나세루 히토와 이마스카 |
| 번역기를 사용해도 될까요? | 翻訳アプリを使ってもいいですか。<br>(혼야쿠 아푸리오 츠캇테모 이이데스카) | 혼야쿠 아푸리오 츠캇테모 이이데스카 |

## 휴대폰에 저장해둘 문장

```text
すみません。日本語があまり話せません。(스미마셍. 니혼고가 아마리 하나세마셍)
この画面を見てください。助けてください。(코노 가멘오 미테 쿠다사이. 타스케테 쿠다사이)

실례합니다. 일본어를 잘 못합니다.
이 화면을 봐주세요. 도와주세요.
```
