# 긴급 상황 일본어

상위 노트: [[여행 일본어 회화]]
관련 노트: [[길 묻기와 교통 일본어]], [[숙소와 체크인 일본어]]

## 바로 써야 하는 말

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 도와주세요 | 助けてください。 | 타스케테 쿠다사이 |
| 경찰을 불러주세요 | 警察を呼んでください。 | 케이사츠오 욘데 쿠다사이 |
| 구급차를 불러주세요 | 救急車を呼んでください。 | 큐-큐-샤오 욘데 쿠다사이 |
| 병원에 가고 싶습니다 | 病院に行きたいです。 | 뵤-인니 이키타이 데스 |
| 길을 잃었습니다 | 道に迷いました。 | 미치니 마요이마시타 |
| 휴대폰을 잃어버렸습니다 | 携帯をなくしました。 | 케이타이오 나쿠시마시타 |
| 지갑을 잃어버렸습니다 | 財布をなくしました。 | 사이후오 나쿠시마시타 |

## 몸 상태 말하기

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 아픕니다 | 痛いです。 | 이타이 데스 |
| 머리가 아픕니다 | 頭が痛いです。 | 아타마가 이타이 데스 |
| 배가 아픕니다 | お腹が痛いです。 | 오나카가 이타이 데스 |
| 열이 있습니다 | 熱があります。 | 네츠가 아리마스 |
| 약이 필요합니다 | 薬が必要です。 | 쿠스리가 히츠요- 데스 |
| 알레르기가 있습니다 | アレルギーがあります。 | 아레루기-가 아리마스 |

## 말이 안 통할 때

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 일본어를 잘 못합니다 | 日本語があまり話せません。 | 니혼고가 아마리 하나세마셍 |
| 한국어 가능한 분 있나요? | 韓国語を話せる人はいますか。 | 칸코쿠고오 하나세루 히토와 이마스카 |
| 영어 가능한 분 있나요? | 英語を話せる人はいますか。 | 에-고오 하나세루 히토와 이마스카 |
| 번역기를 사용해도 될까요? | 翻訳アプリを使ってもいいですか。 | 혼야쿠 아푸리오 츠캇테모 이이데스카 |

## 휴대폰에 저장해둘 문장

```text
すみません。日本語があまり話せません。
この画面を見てください。助けてください。

실례합니다. 일본어를 잘 못합니다.
이 화면을 봐주세요. 도와주세요.
```
