# 가격 묻기와 계산하기 일본어

상위 노트: [[여행 일본어 회화]]
관련 노트: [[일본어 숫자와 금액 듣기]], [[편의점과 식당 일본어]]

## 가격 묻기

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 얼마인가요? | いくらですか。 | 이쿠라 데스카 |
| 이것은 얼마인가요? | これはいくらですか。 | 코레와 이쿠라 데스카 |
| 전부 얼마인가요? | 全部でいくらですか。 | 젠부데 이쿠라 데스카 |
| 더 싼 것은 있나요? | もっと安いのはありますか。 | 못토 야스이노와 아리마스카 |

## 주문/구매하기

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 이것 주세요 | これをください。 | 코레오 쿠다사이 |
| 이것 하나 주세요 | これをひとつください。 | 코레오 히토츠 쿠다사이 |
| 이것 두 개 주세요 | これをふたつください。 | 코레오 후타츠 쿠다사이 |
| 괜찮습니다/필요 없습니다 | 大丈夫です。 | 다이죠-부 데스 |
| 봉투 주세요 | 袋をください。 | 후쿠로오 쿠다사이 |
| 영수증 주세요 | レシートをください。 | 레시-토오 쿠다사이 |

## 결제하기

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 카드로 부탁드립니다 | カードでお願いします。 | 카-도데 오네가이시마스 |
| 현금으로 하겠습니다 | 現金でお願いします。 | 겐킨데 오네가이시마스 |
| 교통카드 되나요? | ICカードは使えますか。 | 아이시- 카-도와 츠카에마스카 |
| 결제해 주세요 | お会計お願いします。 | 오카이케이 오네가이시마스 |

## 계산대에서 들릴 수 있는 질문

| 일본어 | 읽는 법 | 뜻 | 대답 |
|---|---|---|---|
| 袋は要りますか。 | 후쿠로와 이리마스카 | 봉투 필요하세요? | はい / いいえ |
| レシートは要りますか。 | 레시-토와 이리마스카 | 영수증 필요하세요? | はい / いいえ |
| ポイントカードはありますか。 | 포인토 카-도와 아리마스카 | 포인트 카드 있나요? | ありません |
| 温めますか。 | 아타타메마스카 | 데워드릴까요? | はい / いいえ |

## 제일 유용한 대답 5개

| 일본어 | 읽는 법 | 뜻 |
|---|---|---|
| はい | 하이 | 네 |
| いいえ | 이이에 | 아니요 |
| お願いします | 오네가이시마스 | 부탁드립니다 |
| 大丈夫です | 다이죠-부 데스 | 괜찮습니다 |
| ありません | 아리마셍 | 없습니다 |

## 실전 예시

```text
A: これはいくらですか。
   코레와 이쿠라 데스카.
   이것은 얼마인가요?

B: せんにひゃくえんです。
   센니햐쿠엔 데스.
   1,200엔입니다.

A: これをひとつください。カードでお願いします。
   코레오 히토츠 쿠다사이. 카-도데 오네가이시마스.
   이것 하나 주세요. 카드로 부탁드립니다.
```
