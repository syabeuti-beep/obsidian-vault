# 길 묻기와 교통 일본어

상위 노트: [[여행 일본어 회화]]
관련 노트: [[일본어 숫자와 금액 듣기]], [[긴급 상황 일본어]]

## 길 묻기

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 여기는 어디인가요? | ここはどこですか。 | 코코와 도코 데스카 |
| ~는 어디인가요? | 〜はどこですか。 | ~와 도코 데스카 |
| 역은 어디인가요? | 駅はどこですか。 | 에키와 도코 데스카 |
| 버스 정류장은 어디인가요? | バス停はどこですか。 | 바스테-와 도코 데스카 |
| 화장실은 어디인가요? | トイレはどこですか。 | 토이레와 도코 데스카 |
| 이 주소로 가고 싶습니다 | この住所に行きたいです。 | 코노 주-쇼니 이키타이 데스 |

## 방향 표현

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 오른쪽 | 右 | 미기 |
| 왼쪽 | 左 | 히다리 |
| 직진 | まっすぐ | 맛스구 |
| 앞 | 前 | 마에 |
| 뒤 | 後ろ | 우시로 |
| 근처 | 近く | 치카쿠 |
| 멀다 | 遠い | 토오이 |

## 교통

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 기차 | 電車 | 덴샤 |
| 역 | 駅 | 에키 |
| 버스 | バス | 바스 |
| 택시 | タクシー | 타쿠시- |
| 표 | 切符 | 킷푸 |
| 승강장 | 乗り場 | 노리바 |
| 시간표 | 時刻表 | 지코쿠효- |

## 교통 질문

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 이 기차는 ~에 가나요? | この電車は〜に行きますか。 | 코노 덴샤와 ~니 이키마스카 |
| 몇 시에 출발하나요? | 何時に出ますか。 | 난지니 데마스카 |
| 어디서 타나요? | どこで乗りますか。 | 도코데 노리마스카 |
| 어디서 내리나요? | どこで降りますか。 | 도코데 오리마스카 |
| 택시를 불러주세요 | タクシーを呼んでください。 | 타쿠시-오 욘데 쿠다사이 |

## 시간이 급할 때 보여주기 좋은 문장

```text
すみません。この住所に行きたいです。
스미마셍. 코노 주-쇼니 이키타이 데스.
실례합니다. 이 주소로 가고 싶습니다.
```

휴대폰 지도 화면과 같이 보여주면 됩니다.
