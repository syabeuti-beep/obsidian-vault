# 일본어 여행 예절 표현

상위 노트: [[여행 일본어 회화]]
관련 노트: [[편의점과 식당 일본어]], [[숙소와 체크인 일본어]], [[가격 묻기와 계산하기 일본어]]

## 기본 인사

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 안녕하세요 | こんにちは<br>(콘니치와) | 콘니치와 |
| 감사합니다 | ありがとうございます<br>(아리가토- 고자이마스) | 아리가토- 고자이마스 |
| 죄송합니다/실례합니다 | すみません<br>(스미마셍) | 스미마셍 |
| 미안합니다 | ごめんなさい<br>(고멘나사이) | 고멘나사이 |
| 부탁드립니다 | お願いします<br>(오네가이시마스) | 오네가이시마스 |
| 괜찮습니다 | 大丈夫です<br>(다이죠-부 데스) | 다이죠-부 데스 |

## 여행에서 가장 많이 쓰는 짧은 대답

| 일본어 | 읽는 법 | 뜻 |
|---|---|---|
| はい<br>(하이) | 하이 | 네 |
| いいえ<br>(이이에) | 이이에 | 아니요 |
| あります<br>(아리마스) | 아리마스 | 있습니다 |
| ありません<br>(아리마셍) | 아리마셍 | 없습니다 |
| わかりました<br>(와카리마시타) | 와카리마시타 | 알겠습니다 |
| わかりません<br>(와카리마셍) | 와카리마셍 | 모르겠습니다 |

## 부탁을 부드럽게 하기

일본에서는 짧게 명령처럼 말하기보다 `お願いします`, `ください`를 붙이면 안전합니다.

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 이것 부탁드립니다 | これ、お願いします。<br>(코레, 오네가이시마스) | 코레, 오네가이시마스 |
| 이것 주세요 | これをください。<br>(코레오 쿠다사이) | 코레오 쿠다사이 |
| 조금 기다려주세요 | 少し待ってください。<br>(스코시 맛테 쿠다사이) | 스코시 맛테 쿠다사이 |
| 다시 말해주세요 | もう一度言ってください。<br>(모- 이치도 잇테 쿠다사이) | 모- 이치도 잇테 쿠다사이 |
| 천천히 말해주세요 | ゆっくり話してください。<br>(윳쿠리 하나시테 쿠다사이) | 윳쿠리 하나시테 쿠다사이 |

## 몰라도 되는 말 대신 외울 말

복잡한 문법보다 아래 표현만 익혀도 여행에서는 충분합니다.

```text
すみません。(스미마셍)
스미마셍.
실례합니다.

これをください。(코레오 쿠다사이)
코레오 쿠다사이.
이것 주세요.

ありがとうございます。(아리가토- 고자이마스)
아리가토- 고자이마스.
감사합니다.
```

## 주의

- `すみません(스미마셍)`은 “저기요”, “죄송합니다”, “실례합니다”로 모두 쓸 수 있어서 매우 유용합니다.
- 잘 못 알아들었을 때는 고개만 끄덕이지 말고 `わかりません(와카리마셍)` 또는 `もう一度言ってください(모- 이치도 잇테 쿠다사이)`를 쓰면 좋습니다.
