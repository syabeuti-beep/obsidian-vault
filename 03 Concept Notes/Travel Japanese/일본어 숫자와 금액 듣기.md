# 일본어 숫자와 금액 듣기

상위 노트: [[여행 일본어 회화]]
관련 노트: [[가격 묻기와 계산하기 일본어]], [[편의점과 식당 일본어]]

## 0부터 10까지

| 숫자 | 일본어 | 읽는 법 |
|---:|---|---|
| 0 | ゼロ / れい | 제로 / 레이 |
| 1 | いち | 이치 |
| 2 | に | 니 |
| 3 | さん | 산 |
| 4 | よん / し | 욘 / 시 |
| 5 | ご | 고 |
| 6 | ろく | 로쿠 |
| 7 | なな / しち | 나나 / 시치 |
| 8 | はち | 하치 |
| 9 | きゅう / く | 큐- / 쿠 |
| 10 | じゅう | 주- |

여행에서는 4는 보통 よん, 7은 なな로 듣는다고 생각하면 편합니다.

## 10 단위

| 숫자 | 일본어 | 읽는 법 |
|---:|---|---|
| 10 | じゅう | 주- |
| 20 | にじゅう | 니주- |
| 30 | さんじゅう | 산주- |
| 40 | よんじゅう | 욘주- |
| 50 | ごじゅう | 고주- |
| 60 | ろくじゅう | 로쿠주- |
| 70 | ななじゅう | 나나주- |
| 80 | はちじゅう | 하치주- |
| 90 | きゅうじゅう | 큐-주- |

예: 23 = にじゅうさん, 니주-산

## 100, 1,000, 10,000

| 숫자 | 일본어 | 읽는 법 |
|---:|---|---|
| 100 | ひゃく | 햐쿠 |
| 200 | にひゃく | 니햐쿠 |
| 300 | さんびゃく | 산뱌쿠 |
| 400 | よんひゃく | 욘햐쿠 |
| 500 | ごひゃく | 고햐쿠 |
| 600 | ろっぴゃく | 롯뱌쿠 |
| 700 | ななひゃく | 나나햐쿠 |
| 800 | はっぴゃく | 핫뱌쿠 |
| 900 | きゅうひゃく | 큐-햐쿠 |
| 1,000 | せん | 센 |
| 2,000 | にせん | 니센 |
| 3,000 | さんぜん | 산젠 |
| 8,000 | はっせん | 핫센 |
| 10,000 | いちまん | 이치만 |

주의할 발음:

- 300 = さんびゃく, 산뱌쿠
- 600 = ろっぴゃく, 롯뱌쿠
- 800 = はっぴゃく, 핫뱌쿠
- 3,000 = さんぜん, 산젠
- 8,000 = はっせん, 핫센

## 엔화 금액 듣기

円은 えん, 엔입니다.

| 금액 | 일본어 | 읽는 법 |
|---:|---|---|
| 100円 | ひゃくえん | 햐쿠엔 |
| 250円 | にひゃくごじゅうえん | 니햐쿠 고주-엔 |
| 500円 | ごひゃくえん | 고햐쿠엔 |
| 800円 | はっぴゃくえん | 핫뱌쿠엔 |
| 1,000円 | せんえん | 센엔 |
| 1,200円 | せんにひゃくえん | 센니햐쿠엔 |
| 1,500円 | せんごひゃくえん | 센고햐쿠엔 |
| 2,300円 | にせんさんびゃくえん | 니센 산뱌쿠엔 |
| 3,980円 | さんぜんきゅうひゃくはちじゅうえん | 산젠 큐-햐쿠 하치주-엔 |

## 계산대에서 자주 들리는 말

| 일본어 | 읽는 법 | 뜻 |
|---|---|---|
| お会計 | 오카이케이 | 계산 |
| 合計 | 고-케이 | 합계 |
| 〜円です | ~엔 데스 | ~엔입니다 |
| 現金 | 겐킨 | 현금 |
| カード | 카-도 | 카드 |
| レシート | 레시-토 | 영수증 |

## 빠른 연습

- いくらですか。→ 얼마인가요?
- さんびゃくえんです。→ 300엔입니다.
- せんにひゃくえんです。→ 1,200엔입니다.
- カードでお願いします。→ 카드로 부탁드립니다.
