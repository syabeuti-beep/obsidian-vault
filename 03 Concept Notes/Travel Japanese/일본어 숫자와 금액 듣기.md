# 일본어 숫자와 금액 듣기

상위 노트: [[여행 일본어 회화]]
관련 노트: [[가격 묻기와 계산하기 일본어]], [[편의점과 식당 일본어]]

## 0부터 10까지

| 숫자 | 일본어 | 읽는 법 |
|---:|---|---|
| 0 | ゼロ / れい<br>(제로 / 레이) | 제로 / 레이 |
| 1 | いち<br>(이치) | 이치 |
| 2 | に<br>(니) | 니 |
| 3 | さん<br>(산) | 산 |
| 4 | よん / し<br>(욘 / 시) | 욘 / 시 |
| 5 | ご<br>(고) | 고 |
| 6 | ろく<br>(로쿠) | 로쿠 |
| 7 | なな / しち<br>(나나 / 시치) | 나나 / 시치 |
| 8 | はち<br>(하치) | 하치 |
| 9 | きゅう / く<br>(큐- / 쿠) | 큐- / 쿠 |
| 10 | じゅう<br>(주-) | 주- |

여행에서는 4는 보통 よん(욘), 7은 なな(나나)로 듣는다고 생각하면 편합니다.

| 숫자 | 일본어 | 읽는 법 |
|---:|---|---|
| 10 | じゅう<br>(주-) | 주- |
| 20 | にじゅう<br>(니주-) | 니주- |
| 30 | さんじゅう<br>(산주-) | 산주- |
| 40 | よんじゅう<br>(욘주-) | 욘주- |
| 50 | ごじゅう<br>(고주-) | 고주- |
| 60 | ろくじゅう<br>(로쿠주-) | 로쿠주- |
| 70 | ななじゅう<br>(나나주-) | 나나주- |
| 80 | はちじゅう<br>(하치주-) | 하치주- |
| 90 | きゅうじゅう<br>(큐-주-) | 큐-주- |

예: 23 = にじゅうさん(니주-산)

## 100, 1,000, 10,000

| 숫자 | 일본어 | 읽는 법 |
|---:|---|---|
| 100 | ひゃく<br>(햐쿠) | 햐쿠 |
| 200 | にひゃく<br>(니햐쿠) | 니햐쿠 |
| 300 | さんびゃく<br>(산뱌쿠) | 산뱌쿠 |
| 400 | よんひゃく<br>(욘햐쿠) | 욘햐쿠 |
| 500 | ごひゃく<br>(고햐쿠) | 고햐쿠 |
| 600 | ろっぴゃく<br>(롯뱌쿠) | 롯뱌쿠 |
| 700 | ななひゃく<br>(나나햐쿠) | 나나햐쿠 |
| 800 | はっぴゃく<br>(핫뱌쿠) | 핫뱌쿠 |
| 900 | きゅうひゃく<br>(큐-햐쿠) | 큐-햐쿠 |
| 1,000 | せん<br>(센) | 센 |
| 2,000 | にせん<br>(니센) | 니센 |
| 3,000 | さんぜん<br>(산젠) | 산젠 |
| 8,000 | はっせん<br>(핫센) | 핫센 |
| 10,000 | いちまん<br>(이치만) | 이치만 |

주의할 발음:

- 300 = さんびゃく(산뱌쿠)
- 600 = ろっぴゃく(롯뱌쿠)
- 800 = はっぴゃく(핫뱌쿠)
- 3,000 = さんぜん(산젠)
- 8,000 = はっせん(핫센)

## 엔화 금액 듣기

円은 えん(엔)입니다.

| 금액 | 일본어 | 읽는 법 |
|---:|---|---|
| 100円 | ひゃくえん<br>(햐쿠엔) | 햐쿠엔 |
| 250円 | にひゃくごじゅうえん<br>(니햐쿠 고주-엔) | 니햐쿠 고주-엔 |
| 500円 | ごひゃくえん<br>(고햐쿠엔) | 고햐쿠엔 |
| 800円 | はっぴゃくえん<br>(핫뱌쿠엔) | 핫뱌쿠엔 |
| 1,000円 | せんえん<br>(센엔) | 센엔 |
| 1,200円 | せんにひゃくえん<br>(센니햐쿠엔) | 센니햐쿠엔 |
| 1,500円 | せんごひゃくえん<br>(센고햐쿠엔) | 센고햐쿠엔 |
| 2,300円 | にせんさんびゃくえん<br>(니센 산뱌쿠엔) | 니센 산뱌쿠엔 |
| 3,980円 | さんぜんきゅうひゃくはちじゅうえん<br>(산젠 큐-햐쿠 하치주-엔) | 산젠 큐-햐쿠 하치주-엔 |

## 계산대에서 자주 들리는 말

| 일본어 | 읽는 법 | 뜻 |
|---|---|---|
| お会計<br>(오카이케이) | 오카이케이 | 계산 |
| 合計<br>(고-케이) | 고-케이 | 합계 |
| 〜円です<br>(~엔 데스) | ~엔 데스 | ~엔입니다 |
| 現金<br>(겐킨) | 겐킨 | 현금 |
| カード<br>(카-도) | 카-도 | 카드 |
| レシート<br>(레시-토) | 레시-토 | 영수증 |

## 빠른 연습

- いくらですか。(이쿠라 데스카) → 얼마인가요?
- さんびゃくえんです。(산뱌쿠엔 데스) → 300엔입니다.
- せんにひゃくえんです。(센니햐쿠엔 데스) → 1,200엔입니다.
- カードでお願いします。(카-도데 오네가이시마스) → 카드로 부탁드립니다.
