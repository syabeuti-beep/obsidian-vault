# 여행 일본어 회화

## 목적

일본의 외진 지역을 여행할 때 영어가 잘 통하지 않는 상황을 대비해서, 비행기 안에서 빠르게 훑을 수 있는 최소한의 일본어 회화 그래프입니다.

이 노트가 유일한 허브입니다. 아래 노트들은 모두 이 허브에 연결된 하나의 그래프 안에 있습니다.


## 표기 원칙

히라가나와 가타카나를 아직 읽지 못하는 사람을 기준으로 정리했습니다. 일본어 표현 바로 옆에 괄호로 한글 발음을 붙이고, 표에서는 `일본어(발음)` 형태로 먼저 보이게 했습니다.

예: すみません(스미마셍), トイレ(토이레), カード(카-도)

## 그래프 구조

```text
여행 일본어 회화
├─ [[일본어 숫자와 금액 듣기]]
├─ [[가격 묻기와 계산하기 일본어]]

├─ [[길 묻기와 교통 일본어]]
├─ [[숙소와 체크인 일본어]]
├─ [[긴급 상황 일본어]]
└─ [[일본어 여행 예절 표현]]
```

## 비행기에서 보는 추천 순서

1. [[일본어 여행 예절 표현]]: 먼저 `すみません(스미마셍)`, `ありがとうございます(아리가토- 고자이마스)`, `お願いします(오네가이시마스)`만 외우기
2. [[일본어 숫자와 금액 듣기]]: 100엔, 500엔, 1000엔, 1200엔 알아듣기
3. [[가격 묻기와 계산하기 일본어]]: 물건을 사고 계산하기
4. [[편의점과 식당 일본어]]: 편의점/식당에서 바로 쓰기
5. [[길 묻기와 교통 일본어]]: 길을 잃었을 때 지도와 같이 보여주기
6. [[숙소와 체크인 일본어]]: 체크인, 와이파이, 방 문제 말하기
7. [[긴급 상황 일본어]]: 아프거나 잃어버렸을 때 보여주기

## 가장 먼저 외울 10문장

| 상황 | 일본어 | 읽는 법 | 뜻 |
|---|---|---|---|
| 실례합니다 | すみません<br>(스미마셍) | 스미마셍 | 죄송합니다/저기요 |
| 감사합니다 | ありがとうございます<br>(아리가토- 고자이마스) | 아리가토- 고자이마스 | 감사합니다 |
| 이것 주세요 | これをください<br>(코레오 쿠다사이) | 코레오 쿠다사이 | 이것 주세요 |
| 얼마인가요? | いくらですか<br>(이쿠라 데스카) | 이쿠라 데스카 | 얼마예요? |
| 어디인가요? | どこですか<br>(도코 데스카) | 도코 데스카 | 어디예요? |
| 화장실은 어디인가요? | トイレはどこですか<br>(토이레와 도코 데스카) | 토이레와 도코 데스카 | 화장실 어디예요? |
| 영어 할 수 있나요? | 英語は話せますか<br>(에-고와 하나세마스카) | 에-고와 하나세마스카 | 영어 가능하세요? |
| 잘 모르겠습니다 | わかりません<br>(와카리마셍) | 와카리마셍 | 모르겠습니다 |
| 천천히 말해주세요 | ゆっくり話してください<br>(윳쿠리 하나시테 쿠다사이) | 윳쿠리 하나시테 쿠다사이 | 천천히 말해주세요 |
| 도와주세요 | 助けてください<br>(타스케테 쿠다사이) | 타스케테 쿠다사이 | 도와주세요 |

## 손짓과 같이 쓰면 좋은 기본 구조

- 이것: これ(코레)
- 저것: あれ(아레)
- 여기: ここ(코코)
- 저기: あそこ(아소코)
- 하나: ひとつ(히토츠)
- 두 개: ふたつ(후타츠)

예시:

```text
これをひとつください。(코레오 히토츠 쿠다사이)
코레오 히토츠 쿠다사이.
이것 하나 주세요.
```

## 바로 보여줄 수 있는 문장

```text
すみません。日本語があまり話せません。(스미마셍. 니혼고가 아마리 하나세마셍)
ゆっくり話してください。(윳쿠리 하나시테 쿠다사이)

실례합니다. 일본어를 잘 못합니다.
천천히 말해주세요.
```
