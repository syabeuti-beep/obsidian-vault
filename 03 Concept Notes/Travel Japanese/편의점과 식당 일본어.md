# 편의점과 식당 일본어

상위 노트: [[여행 일본어 회화]]
관련 노트: [[가격 묻기와 계산하기 일본어]], [[일본어 숫자와 금액 듣기]], [[일본어 여행 예절 표현]]

## 식당 들어갈 때

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 한 명입니다 | 一人です。<br>(히토리 데스) | 히토리 데스 |
| 두 명입니다 | 二人です。<br>(후타리 데스) | 후타리 데스 |
| 예약했습니다 | 予約しました。<br>(요야쿠 시마시타) | 요야쿠 시마시타 |
| 예약 안 했습니다 | 予約していません。<br>(요야쿠 시테이마셍) | 요야쿠 시테이마셍 |
| 메뉴 주세요 | メニューをください。<br>(메뉴-오 쿠다사이) | 메뉴-오 쿠다사이 |

## 주문하기

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 이것 주세요 | これをください。<br>(코레오 쿠다사이) | 코레오 쿠다사이 |
| 추천 메뉴가 뭐예요? | おすすめは何ですか。<br>(오스스메와 난데스카) | 오스스메와 난데스카 |
| 물 주세요 | 水をください。<br>(미즈오 쿠다사이) | 미즈오 쿠다사이 |
| 따뜻한 차 주세요 | 温かいお茶をください。<br>(아타타카이 오차오 쿠다사이) | 아타타카이 오차오 쿠다사이 |
| 맵지 않게 해주세요 | 辛くしないでください。<br>(카라쿠 시나이데 쿠다사이) | 카라쿠 시나이데 쿠다사이 |
| 고기 빼주세요 | 肉を抜いてください。<br>(니쿠오 누이테 쿠다사이) | 니쿠오 누이테 쿠다사이 |

## 계산하기

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 계산 부탁드립니다 | お会計お願いします。<br>(오카이케이 오네가이시마스) | 오카이케이 오네가이시마스 |
| 따로 계산할 수 있나요? | 別々にできますか。<br>(베츠베츠니 데키마스카) | 베츠베츠니 데키마스카 |
| 카드 되나요? | カードは使えますか。<br>(카-도와 츠카에마스카) | 카-도와 츠카에마스카 |
| 잘 먹었습니다 | ごちそうさまでした。<br>(고치소-사마데시타) | 고치소-사마데시타 |

## 편의점에서 자주 들리는 말

| 일본어 | 읽는 법 | 뜻 | 대답 |
|---|---|---|---|
| 温めますか。<br>(아타타메마스카) | 아타타메마스카 | 데워드릴까요? | はい / いいえ |
| 袋は要りますか。<br>(후쿠로와 이리마스카) | 후쿠로와 이리마스카 | 봉투 필요하세요? | はい / いいえ |
| お箸は要りますか。<br>(오하시와 이리마스카) | 오하시와 이리마스카 | 젓가락 필요하세요? | はい / いいえ |
| スプーンは要りますか。<br>(스푼와 이리마스카) | 스푼와 이리마스카 | 숟가락 필요하세요? | はい / いいえ |
| ポイントカードはありますか。<br>(포인토 카-도와 아리마스카) | 포인토 카-도와 아리마스카 | 포인트 카드 있나요? | ありません |

## 알레르기/못 먹는 음식

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 알레르기가 있습니다 | アレルギーがあります。<br>(아레루기-가 아리마스) | 아레루기-가 아리마스 |
| 해산물을 못 먹습니다 | 海鮮が食べられません。<br>(카이센가 타베라레마셍) | 카이센가 타베라레마셍 |
| 고기를 못 먹습니다 | 肉が食べられません。<br>(니쿠가 타베라레마셍) | 니쿠가 타베라레마셍 |
| 술은 못 마십니다 | お酒は飲めません。<br>(오사케와 노메마셍) | 오사케와 노메마셍 |

## 실전 한 문장

```text
すみません、これをひとつください。温めてください。(스미마셍, 코레오 히토츠 쿠다사이. 아타타메테 쿠다사이)
스미마셍, 코레오 히토츠 쿠다사이. 아타타메테 쿠다사이.
실례합니다, 이것 하나 주세요. 데워주세요.
```
