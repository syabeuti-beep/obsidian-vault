# 숙소와 체크인 일본어

상위 노트: [[여행 일본어 회화]]
관련 노트: [[가격 묻기와 계산하기 일본어]], [[긴급 상황 일본어]], [[일본어 여행 예절 표현]]

## 체크인

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 체크인 부탁드립니다 | チェックインお願いします。 | 첵쿠인 오네가이시마스 |
| 예약했습니다 | 予約しました。 | 요야쿠 시마시타 |
| 제 이름은 ~입니다 | 名前は〜です。 | 나마에와 ~데스 |
| 여권입니다 | パスポートです。 | 파스포-토 데스 |
| 몇 시에 체크아웃인가요? | チェックアウトは何時ですか。 | 첵쿠아우토와 난지 데스카 |

## 방/시설 관련

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 와이파이 비밀번호는 무엇인가요? | Wi-Fiのパスワードは何ですか。 | 와이파이노 파스와-도와 난데스카 |
| 방은 어디인가요? | 部屋はどこですか。 | 헤야와 도코 데스카 |
| 열쇠/카드키 | 鍵 / カードキー | 카기 / 카-도키- |
| 뜨거운 물이 안 나와요 | お湯が出ません。 | 오유가 데마셍 |
| 에어컨이 안 돼요 | エアコンが動きません。 | 에아콘가 우고키마셍 |

## 부탁하기

| 한국어 | 일본어 | 읽는 법 |
|---|---|---|
| 도와주세요 | 手伝ってください。 | 테츠닷테 쿠다사이 |
| 다시 말해주세요 | もう一度言ってください。 | 모- 이치도 잇테 쿠다사이 |
| 천천히 말해주세요 | ゆっくり話してください。 | 윳쿠리 하나시테 쿠다사이 |
| 종이에 써주세요 | 紙に書いてください。 | 카미니 카이테 쿠다사이 |

## 실전 예시

```text
予約しました。名前はホジンです。
요야쿠 시마시타. 나마에와 호진 데스.
예약했습니다. 이름은 호진입니다.
```
