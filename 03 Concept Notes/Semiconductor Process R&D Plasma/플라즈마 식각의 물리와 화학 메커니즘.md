# 플라즈마 식각의 물리와 화학 메커니즘

## 핵심 문장

플라즈마 식각은 보통 **화학 반응만으로도, 물리적 sputtering만으로도 충분하지 않고**, 두 효과가 함께 작동할 때 좋은 속도와 profile을 얻습니다.

## 세 가지 극한

| 방식 | 설명 | 장점 | 한계 |
|---|---|---|---|
| 순수 화학 식각 | 라디칼이 표면과 반응해 휘발성 부산물 생성 | 선택비가 좋을 수 있음 | 등방성, undercut |
| 순수 물리 식각 | 이온 충돌로 원자를 때려냄 | 방향성 좋음 | 느림, 손상 큼, 선택비 낮음 |
| 이온 보조 화학 식각 | 이온이 표면 결합을 깨고 라디칼 반응을 촉진 | 속도와 방향성 균형 | 제어 변수 많음 |

## 표면에서 일어나는 단계

1. 반응종이 표면으로 이동
2. 표면에 흡착
3. 이온 충돌이 결합을 약화하거나 표면을 활성화
4. 라디칼과 표면 원자가 반응
5. 휘발성 부산물이 탈착
6. 펌프로 배출

이 중 하나라도 막히면 식각률이 낮아지거나 residue가 생깁니다.

## 휘발성 부산물이 중요한 이유

식각은 “재료를 반응시킨 뒤 밖으로 빼내는” 과정입니다. 반응 생성물이 휘발성이 낮으면 표면에 남아 다음 식각을 방해합니다.

예를 들어 어떤 재료는 F radical과 만나 휘발성 fluorides를 만들기 쉬워 잘 식각되고, 어떤 재료는 더 어려워 다른 chemistry가 필요합니다.

## Polymer passivation의 양면성

Fluorocarbon 계열 식각에서는 표면이나 옆벽에 polymer가 생길 수 있습니다.

- 장점: 옆벽을 보호해 수직 profile 유지
- 단점: 과하면 바닥 식각 방해, residue 증가

따라서 산화막 식각 등에서는 “깎는 성분”과 “보호막을 만드는 성분”의 균형이 중요합니다.

## Profile을 결정하는 경쟁

```text
수직 식각을 키우는 요인:
- 이온 에너지
- 낮은 압력의 방향성
- 바닥에서의 이온 보조 반응

옆벽 보호/변형 요인:
- polymer passivation
- 라디칼의 등방성 반응
- charging과 이온 궤적 변화
```

## 고종횡비 구조에서 어려운 점

DRAM/NAND 공정에서는 깊고 좁은 구조를 식각해야 하는 경우가 많습니다. 이때 입자가 깊은 trench/hole 바닥까지 도달하기 어렵고, charging과 부산물 배출 문제가 커집니다.

대표 문제:

- aspect ratio dependent etching: 구조가 깊을수록 식각률 감소
- bowing: 중간 옆벽이 넓어짐
- mask erosion: 마스크가 먼저 닳음
- bottom residue: 바닥 잔류물

## 공정 해석 질문 템플릿

문제가 생겼을 때 다음 순서로 묻습니다.

1. 화학 반응종이 부족한가?
2. 이온 에너지가 부족하거나 과한가?
3. 부산물이 휘발/배출되지 못하는가?
4. 옆벽 보호막이 부족하거나 과한가?
5. 웨이퍼 위치별 plasma/gas/temperature 균일도가 다른가?
