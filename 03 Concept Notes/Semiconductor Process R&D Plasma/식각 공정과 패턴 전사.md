# 식각 공정과 패턴 전사

## 한 줄 정의

식각은 포토리소그래피로 만든 마스크 패턴을 이용해, 아래 재료를 원하는 모양과 깊이로 제거하는 공정입니다.

## 식각이 필요한 이유

포토 공정은 감광막에 “그림”을 그리는 단계입니다. 하지만 실제 소자 구조는 산화막, 질화막, 실리콘, 금속 같은 재료에 형성되어야 합니다. 따라서 포토 패턴을 실제 막으로 옮기는 **패턴 전사**가 필요합니다.

## 습식 식각과 건식 식각

| 구분 | 습식 식각 | 건식 식각 |
|---|---|---|
| 매질 | 액체 화학 용액 | 기체/플라즈마 |
| 장점 | 빠르고 단순, 선택비가 좋을 수 있음 | 미세 패턴, 방향성 제어 가능 |
| 단점 | 등방성이라 옆으로 파고들기 쉬움 | 장비/플라즈마 제어가 복잡 |
| 미세공정 적합성 | 제한적 | 매우 중요 |

나노미터급 패턴에서는 옆으로 파고드는 식각을 줄이고 수직으로 깎아야 하므로, 플라즈마를 이용한 건식 식각이 핵심이 됩니다.

## 등방성 vs 이방성

- **등방성 식각**: 모든 방향으로 비슷하게 깎임
- **이방성 식각**: 주로 수직 방향으로 깎임

플라즈마 식각의 강점은 이온이 sheath 전기장에 의해 웨이퍼 방향으로 가속되면서 **방향성**을 제공한다는 점입니다.

```text
이상적 이방성 식각

마스크:  █████   █████
막:      █████████████
식각 후: █████   █████
          ↓       ↓
        수직 방향 제거
```

## 선택비의 의미

식각은 목표막만 깎고, 마스크와 하지막은 덜 깎아야 합니다.

```text
선택비 = 목표막 식각률 / 보호해야 할 막의 식각률
```

예를 들어 산화막을 100 nm/min으로 깎고, Si 하지막이 2 nm/min으로 깎이면 선택비는 50:1입니다.

## endpoint와 overetch

실제 웨이퍼는 완벽히 균일하지 않기 때문에 목표 두께만큼 정확히 식각하고 끝내면 일부 영역은 덜 깎일 수 있습니다. 그래서 endpoint를 감지한 뒤 약간 더 식각하는 **overetch**를 둡니다.

문제는 overetch가 너무 크면 하지막 손상과 CD 변화가 커진다는 점입니다.

## 공정 엔지니어가 보는 대표 결함

| 결함 | 설명 | 원인 예시 |
|---|---|---|
| Undercut | 마스크 아래 옆방향으로 깎임 | 화학 반응이 너무 등방적 |
| Bowing | 깊은 구조 옆벽이 휘어짐 | 이온 각도 분포, charging |
| Microtrenching | 바닥 모서리 과식각 | 이온 반사/전기장 효과 |
| Notching | 특정 계면 근처 lateral etch | charging, 하지막 영향 |
| Residue | 식각 부산물 잔류 | 휘발성 부족, polymer 과다 |
