# 플라즈마 기초를 공정 언어로 번역하기

## 출발점

학부 플라즈마 이론에서 배운 내용은 공정 식각을 이해하는 데 매우 유용합니다. 다만 공정에서는 수식 자체보다 다음 번역이 중요합니다.

> 플라즈마 변수 → 웨이퍼에 도달하는 이온 에너지, 이온 flux, 라디칼 flux, 표면 반응 확률 → 식각 결과

## 핵심 개념 번역표

| 플라즈마 이론 용어 | 공정 언어 | 식각 결과와의 연결 |
|---|---|---|
| 전자온도 | 전자가 가스를 얼마나 잘 해리/이온화하는가 | 라디칼/이온 생성량 |
| 전자밀도 | 플라즈마 밀도 | 이온 flux, 식각률 |
| Sheath | 웨이퍼 근처 전위 강하 영역 | 이온 가속, 방향성 |
| Bias voltage | 웨이퍼에 걸리는 RF bias | 이온 에너지 조절 |
| 평균자유행로 | 입자가 충돌 없이 이동하는 거리 | 이온 각도 분포, profile |
| Debye length | 전기장이 차폐되는 길이 | sheath 구조 이해 |
| 플라즈마 전위 | bulk plasma의 기준 전위 | 이온이 얻는 에너지와 관련 |

## 왜 전자는 화학을 만들고, 이온은 방향성을 만드는가

전자들은 가볍고 RF 전기장에 민감하게 반응합니다. 전자가 중성 가스와 충돌하면 해리와 이온화가 일어납니다.

```text
CF4 + e- -> CF3 + F + e-
CF4 + e- -> CF3+ + F + 2e-
```

여기서 F radical 같은 반응성 중성종은 표면에서 화학 반응을 통해 재료를 제거합니다.

반면 이온은 양전하를 띠고 sheath 전기장에서 웨이퍼 방향으로 가속됩니다. 이 방향성이 수직 식각을 돕습니다.

## 라디칼 flux와 이온 flux

| 항목 | 라디칼 | 이온 |
|---|---|---|
| 전하 | 대체로 중성 | 양전하 |
| 역할 | 화학 반응 | 방향성 충돌, 표면 활성화 |
| 방향성 | 약함 | 강함 |
| 과하면 | 등방성 식각, undercut | damage, sputtering, charging |

좋은 식각은 둘 중 하나만 강한 것이 아니라, **라디칼 화학과 이온 보조 효과의 균형**입니다.

## 압력이 왜 중요한가

압력이 낮으면 평균자유행로가 길어져 이온이 충돌 없이 웨이퍼에 도달하기 쉽습니다. 그러면 방향성이 좋아집니다. 압력이 높으면 충돌이 많아져 라디칼 생성이나 균일성은 좋아질 수 있지만, 이온 방향성이 약해질 수 있습니다.

## Toy 예시

같은 가스 조성에서 bias power만 올렸다고 생각합니다.

- 이온 에너지 증가
- 수직 방향 식각 강화
- hard mask나 하지막 손상 가능성 증가
- CD 변화와 roughness가 증가할 수 있음

따라서 “bias를 올리면 더 잘 깎인다”가 아니라, **식각률/profile/손상/선택비의 trade-off**로 봐야 합니다.
