# 플라즈마 식각 장비와 조절 변수

## 한 줄 정의

플라즈마 식각 장비는 진공 챔버 안에서 반응성 가스를 플라즈마로 만들고, 웨이퍼에 도달하는 라디칼과 이온의 양·에너지·방향을 조절해 재료를 깎는 장비입니다.

## 기본 구성

| 구성 | 역할 |
|---|---|
| Chamber | 진공과 플라즈마가 형성되는 공간 |
| Gas line/MFC | CF계, Cl계, O2, Ar 등 가스 유량 제어 |
| RF source power | 플라즈마 밀도와 radical/ion 생성 조절 |
| RF bias power | 웨이퍼 쪽 이온 에너지 조절 |
| ESC | 웨이퍼 고정과 온도 제어 |
| Pump/throttle valve | 압력 제어 |
| Endpoint detector | 특정 막이 끝나는 시점 감지 |

## CCP와 ICP의 직관적 차이

| 장비 타입 | 직관 | 특징 |
|---|---|---|
| CCP | 전극 사이 RF 전기장으로 플라즈마 형성 | 구조 단순, bias와 plasma가 결합되기 쉬움 |
| ICP | 코일 유도 전기장으로 고밀도 플라즈마 형성 | source power와 bias power를 더 분리해 제어 가능 |

공정 R&D 관점에서는 장비 이름보다 **source power는 plasma density 쪽, bias power는 ion energy 쪽**이라는 분리된 직관이 중요합니다. 실제 장비에서는 완전히 독립적이지 않고 coupling이 있습니다.

## 주요 조절 변수와 영향

| 변수 | 주로 바꾸는 것 | 기대 효과 | 부작용/주의 |
|---|---|---|---|
| Source power | 플라즈마 밀도, 라디칼 생성 | 식각률 증가 가능 | 과도한 해리, 균일도 변화 |
| Bias power | 이온 에너지 | 이방성, 바닥 식각 강화 | damage, mask erosion |
| Pressure | 충돌 빈도, 평균자유행로 | 균일도/화학 반응 변화 | profile 악화 가능 |
| Gas ratio | 라디칼 종류, polymer 형성 | 선택비/profile 제어 | residue 또는 undercut |
| Temperature | 표면 반응/부산물 휘발 | 반응성, polymer 조절 | 균일도와 재현성 이슈 |
| Time | 총 식각량 | depth 조절 | overetch risk |

## 가스 chemistry의 예시적 직관

| 가스 계열 | 주 용도 직관 |
|---|---|
| Fluorocarbon 계열 | 산화막/질화막 식각, polymer passivation 제어 |
| Chlorine 계열 | Si, 금속성 재료 식각에 자주 사용 |
| Oxygen | PR ash, polymer 제거, 산화 반응 보조 |
| Argon | 물리적 sputtering, plasma 안정화 보조 |

구체 chemistry는 재료와 장비에 따라 달라지므로, 암기보다 “어떤 반응종과 부산물을 만드는가”를 보는 것이 좋습니다.

## Recipe를 보는 법

식각 recipe는 대략 다음 형태입니다.

```text
Step 1: breakthrough 또는 native oxide 제거
Step 2: main etch
Step 3: overetch
Step 4: clean 또는 descum
```

각 step은 gas, pressure, source power, bias power, temperature, time이 다릅니다. 공정 R&D는 이 recipe를 조정해 목표 profile과 전기적 특성을 맞춥니다.
