# T5

## 한 줄 요약

[[T5]]는 Text-to-Text Transfer Transformer로 거의 모든 NLP 문제를 텍스트 입력에서 텍스트 출력으로 통일한 encoder-decoder 모델이다.

---

## 핵심 개념

- 번역, 요약, 분류, 질의응답을 모두 text-to-text 형식으로 다룬다.
- encoder-decoder Transformer 구조를 사용한다.
- 입력에 task prefix를 붙여 모델이 어떤 작업을 해야 하는지 알려준다.

---

## 수학적 형태

예를 들어 분류도 다음처럼 텍스트 생성 문제로 바꾼다.

```text
input: classify sentiment: this movie is great
output: positive
```

---

## 컴퓨팅 관점의 입력과 출력

입력은 task prefix가 붙은 source token IDs `[batch, src_len]`, 출력은 decoder token logits `[batch, tgt_len, vocab_size]`입니다.

---

## 간단한 toy 예시

`summarize: ...긴 문서...`를 입력하면 decoder가 요약 문장을 token by token으로 생성합니다.
